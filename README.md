# ShadowLift

## 概要
ShadowLift は、ブラウザ上で画像のシャドウ補正を行うシンプルな画像調整アプリです。1枚なら詳細編集、複数枚なら一括変換してまとめてダウンロードできます。処理はすべてローカルで行われ、画像はサーバーに送信されません。

## Overview
ShadowLift is a simple, in-browser image adjustment app focused on shadow correction. Edit a single image in detail, or batch-process multiple images and download them together. All processing happens locally in your browser—no images are uploaded.

## 使い方
1. 画像をドラッグ&ドロップまたは選択します。
2. 1枚の場合はエディタで調整します。
3. 複数枚の場合は一括補正してダウンロードします。

## Usage
1. Drag and drop (or select) images.
2. For a single image, adjust in the editor.
3. For multiple images, run batch correction and download.
